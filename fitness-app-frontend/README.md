# Fitness App — Frontend

React + Vite single-page app for the Fitness App. See the [root README](../README.md) for the full system overview, architecture, and environment variable reference.

## Quick Start

```bash
cp .env.example .env   # adjust if not using localhost defaults
npm install
npm run dev             # http://localhost:5173
```

## Scripts

- `npm run dev` — start the Vite dev server
- `npm run build` — production build (output in `dist/`)
- `npm run preview` — preview the production build locally
- `npm run lint` — run Oxlint
